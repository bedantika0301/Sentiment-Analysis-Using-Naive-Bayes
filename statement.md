In today’s world, people constantly share their opinions online through reviews, comments, and social media posts. These opinions can tell us a lot about how people feel—whether they are happy, dissatisfied, or neutral about a product, service, or topic. However, going through thousands of such texts manually to understand the overall sentiment is difficult, time-consuming, and not practical.

The main problem is to create a system that can automatically understand and classify the sentiment of text in a quick and efficient way. The system should be able to take input from a user, process it, and determine whether the sentiment is positive, negative, or neutral, without requiring human effort each time. It should also be simple enough to run easily and give reliable results.

To solve this, this project builds a Sentiment Analysis model using the Naive Bayes algorithm. The system first cleans the input text by converting it to lowercase and removing unnecessary punctuation. Then, it converts the text into numbers using a method called Bag of Words, so that the machine learning model can understand it.

Once trained on sample data, the model can analyze new sentences entered by the user and predict their sentiment. It also provides a confidence score, showing how sure the model is about its prediction.

Although the model uses a small dataset, it clearly shows how machine learning can be used to understand human language. This project is a simple but effective step toward building more advanced systems that can analyze large amounts of text automatically.
